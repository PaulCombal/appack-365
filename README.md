MS 365 AppPack
===

This repository contains the files needed to build a Microsoft 365 [AppPack](https://github.com/PaulCombal/appack-cli).

This repository contains icons in the public domain for various licensed products.
None of the contributors to this repository are affiliated with any company for which icons are present.

Some files are not included in this repository, for technical and legal reasons:
* The guest addons ISO (virtio-win, guest-addons.iso)
* The main OS image (image.qcow2)
* The installation medium (installer.iso)
