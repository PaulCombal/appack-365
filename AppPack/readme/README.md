## Microsoft 365 Application Package

This package contains a comprehensive collection of core **Microsoft 365** applications (formerly known as Microsoft Office).

### Core Applications Included:

* Microsoft Excel
* Microsoft Access
* Microsoft Publisher
* Microsoft OneNote
* Microsoft Outlook (Classic Interface)
* Microsoft PowerPoint
* Microsoft Word

### Supplementary Utilities:

The AppPack also provides access to the following configuration tools:

* **Office Language Settings** Configuration Utility
* **Plain Remote Desktop Protocol (RDP) Session** for advanced, manual configuration
